# STROBE checklist (cross-sectional) – Supplementary File S1

Manuscript: `fire_technology/manuscript.md`

This checklist is completed against the STROBE Statement for cross-sectional studies (von Elm et al., 2007). Locations refer to manuscript section headings.

| STROBE item | Recommendation | Where addressed |
|---:|---|---|
| 1a | Indicate study design in title/abstract | Title; Abstract (Methods) |
| 1b | Provide informative and balanced summary | Abstract (Aims/Methods/Results/Conclusions) |
| 2 | Explain scientific background and rationale | Background |
| 3 | State specific objectives/prespecified hypotheses | Aims |
| 4 | Present key elements of study design early | Abstract (Methods); Methods (Analysis) |
| 5 | Describe setting, locations, relevant dates | Methods (Data source; temporal coverage 1968–2025) |
| 6a | Give eligibility criteria and selection of participants | Methods (Data source; unit of analysis; inclusion) |
| 7 | Clearly define outcomes, exposures, predictors, confounders | Methods (Variables); Limitations (unmeasured determinants) |
| 8 | Data sources/measurement for each variable | Methods (Data source; Variables; population denominators) |
| 9 | Describe efforts to address potential sources of bias | Background (traceability rationale); Methods (source links); Limitations |
| 10 | Explain how study size was arrived at | Methods (included all incidents in export; n=113) |
| 11 | Explain handling of quantitative variables | Results (banding of construction year; decade bins; rate definition) |
| 12a | Describe all statistical methods | Methods (Analysis; descriptive methods; Poisson CI note); Results/Discussion |
| 12b | Methods for subgroups/interactions | Results (post-1996 completeness; Table 3 subset) |
| 12c | Address missing data | Results (construction year missingness; post-1996 completeness); Limitations |
| 12d | Analytical methods accounting for sampling strategy | Not applicable (no sampling; full export analysed) |
| 12e | Sensitivity analyses | Results (Table 3 post-1996 subset) |
| 13a | Report numbers at each stage | Results (Table 1 totals; Table 3 subset size) |
| 13b | Reasons for non-participation at each stage | Not applicable (registry export; no participant recruitment) |
| 13c | Flow diagram | Not applicable |
| 14a | Give characteristics of participants and exposures/confounders | Results (incident counts, categories, seasonality, building bands); Limitations (unmeasured factors) |
| 14b | Indicate number of participants with missing data | Results (construction year missingness) |
| 15 | Report outcome events or summary measures | Results (Tables 1–3; Figures 1–3) |
| 16a | Give unadjusted estimates and precision | Results (crude rates); Discussion (Poisson CI example) |
| 16b | Report category boundaries when continuous variables categorized | Results (construction year bands; decade periods) |
| 16c | Translate estimates into absolute risk (if relevant) | Results/Discussion (deaths per million population) |
| 17 | Report other analyses (subgroups) | Results (Table 3 post-1996 subset; other incident types) |
| 18 | Summarise key results with respect to objectives | Conclusions |
| 19 | Discuss limitations (bias/imprecision) | Limitations |
| 20 | Provide cautious interpretation | Discussion; Conclusions |
| 21 | Discuss generalisability | Discussion (Nordic comparisons; MSB caveat) |
| 22 | Funding and role of funders | Funding |
